# <p align="center">b-ta.DavisDrozReplaceMarvinsFinestHour</p>
<p align="center">Replaces Marvin's Finest Hour faction with Davis & Droz.</p>

A cool side effect is that with Marvin's Finest Hour faction, the voice will match Marvin's ID card, even in the Frontier Defense!

Don't use it with [DavisDrozReplaceFactions](https://northstar.thunderstore.io/package/adventure/Davis_Droz_Replace_Factions)!

_This is an edit of **Cpone**'s DavisDrozReplaceGates._
